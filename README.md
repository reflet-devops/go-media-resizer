# GO media resizer


## Requirements

### Avif

Make sure libaom is installed. On typical Linux distro just run:

#### Debian (and derivatives):
```bash
sudo apt-get install libaom-dev
```

#### RHEL (and derivatives):
```bash
sudo dnf install libaom-devel
```

### Webp

#### MacOS:

```bash
brew install webp
```
#### Linux:

```bash

sudo apt-get install libwebp-dev
```
